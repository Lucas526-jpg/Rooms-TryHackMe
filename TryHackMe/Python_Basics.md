## OPERACIONES MATEMATICAS

| Operacion |	Sintaxis	| Ejemplo |
| :-- | :-- | :-- | 
| Suma	| +	| 1 + 1 = 2 |
| Resta	| -	| 5 - 1 = 4 |
| Producto	| *	| 10 * 10 = 100 |
| Division	| /	| 10 / 2 = 5 |
| Modulo	| %	| 10 % 2 = 0 |
| Potencia	| **	| 5**2 = 25 (5´2) |
